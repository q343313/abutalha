Local Market (E-Commerce App)

A modern full-stack Flutter E-Commerce application with FastAPI backend, providing login/signup, product management, notifications, and cloud integration. Designed with a smooth UI supporting both light and dark modes for sellers and users.

Key Features

  • User authentication (Login / Signup)
  • Role-based access for Users and Shop Owners
  • Product upload with image support
  • Real-time product listings and updates
  • Push notifications (via Firebase Cloud Messaging)
  • Profile editing and account management
  • Light / Dark theme with adaptive UI
  • Owner dashboard with analytics and controls
  • Offline cache and FastAPI backend integration
  • Secure token authentication and REST APIs
