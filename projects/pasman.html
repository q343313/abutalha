Password Manager

A fully secure Flutter-based Password Manager designed to store, encrypt, and protect user credentials locally. It uses modern cryptography, secure local databases (SQLite), and optional FastAPI backend integration for account authentication and remote sync. The app provides a beautiful dark/light UI, biometric unlock, password generator, and real-time notifications.
